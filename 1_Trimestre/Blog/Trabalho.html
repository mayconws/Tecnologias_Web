Propriedades do CSS:

:link

  • que é?

    • O link é um endereço de ligação de um documento ou um conteúdo na web.

  • O que ele faz?

    • O link tem o efeito de ligar uma página a outra, pode-se aplicar efeito de cores na visualização do link.

  • Quais valores podem ser utilizados nesta propriedade?

    • Podem ser utilizados com os valores: color, font-famliy, background.

  • Exemplo de código:

    • <style>

  a {
    color: red;
  }

</style>

:visited

  • O que é?

    • É um seletor para ligações com efeito quando é passado o mouse sobre ele.

  • O que ele faz?

    • Quando o usário passa o muse sobre o link ele muda de aspecto, podendo ficar com uma cor ou tamanho diferente

    sinalizando que o link já foi visitado.

  • Quais valores podem ser utilizados nesta propriedade?

    • Podem ser utilizado com os valores: color, background-color, border-color, outline color, column-rule-color.

  • Exemplo de código:

    • <style>

  a.ex1:visited {
    color: green;
  }

</style>

:hover

  • que é?

    • A função hover é usado para selecionar elementos quando você passa o mouse sobre eles.

  • O que ele faz?

    • Ele muda a cor quando o usuário passa o mouse sobre links, textos, menus entre outros.

  • Quais valores podem ser utilizados nesta propriedade?

    • Podem ser utilizado com os valores: color, background-color, display, text-decoration, font-size.

  • Exemplo de código:

    • <style>

div {
    background-color:green;
    padding: 20px;
    display: none;
}

span:hover + div {
    display: block;
}
</style>

:active

  • que é?

    • A função "active", é usado para selecionar o estilo do link ativo.

  • O que ele faz?

    • Ele torna-se ativo quando você clica nele.

  • Quais valores podem ser utilizados nesta propriedade?

    • Podem ser utilizado com os valores: color, background-color, font-famliy, font-size, text-decoration.

  • Exemplo de código:

    • <style>

    p:active, h1:active, a:active{
        background-color: yellow;
    }

    </style>
