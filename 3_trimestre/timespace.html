Plugin: Jquery Timespace


O que é o Plugin?

Timespace é um plugin jQuery para criar uma linha de tempo dinâmica, responsiva, horizontal, arrastável e sensível ao toque para exibir eventos personalizados de uma maneira deslizante. Clicando no slot do evento é possível ter maiores informações e detalhes sobre o acontecimento.

Como faz para utilizá-lo?

  • Passo 1:

    • Carregue a versão mais recente do jQuery e do código JavaScript do plugin Timespace na sua página HTML.

    <script src="js/jquery-3.3.1.js"></script>

<script src="jquery.timespace.js"></script>

  • Passo 2:

    • Carregue um tema CSS de sua escolha na seção principal da página.

    <!-- Tema Escuro -->
<link href="css/jquery.timespace.dark.css" rel="stylesheet">

<!-- Tema Claro -->
<link href="css/jquery.timespace.light.css" rel="stylesheet">

  • Passo 3:

    • Crie um container onde a linha do tempo do evento deve aparecer.

    <div id="timeline"></div>
